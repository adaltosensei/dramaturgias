# Esquetes Indígenas — Livros-jogo

Três livros-jogo interativos (HTML, sem instalação, funcionam em qualquer navegador),
baseados nas três dramaturgias indígenas contemporâneas usadas como corpus da pesquisa
de mestrado de Adalto Carlos da Conceição (PROFLETRAS – UPE).

- `index.html` — página inicial com o menu para os três jogos
- `siaburu.html` — baseado em Puri & Mara (2022)
- `nossa_luta_e_ancestral.html` — baseado no Teatro Maiuhi (2022)
- `tape_mbyapeha.html` — baseado no Grupo Liberdade PKR (2023)

## Como publicar gratuitamente no GitHub Pages

1. **Crie uma conta gratuita** em [github.com](https://github.com), se ainda não tiver.

2. **Crie um repositório novo**:
   - Clique no `+` no canto superior direito → "New repository".
   - Dê um nome, por exemplo: `esquetes-indigenas-jogos`.
   - Marque como **Public** (necessário para o GitHub Pages gratuito).
   - Clique em "Create repository".

3. **Envie os quatro arquivos** (`index.html`, `siaburu.html`, `nossa_luta_e_ancestral.html`, `tape_mbyapeha.html`):
   - Na página do repositório recém-criado, clique em "uploading an existing file"
     (ou "Add file" → "Upload files").
   - Arraste os quatro arquivos desta pasta para a área de upload.
   - Role até o final e clique em "Commit changes".

4. **Ative o GitHub Pages**:
   - No repositório, vá em **Settings** (aba no topo).
   - No menu lateral esquerdo, clique em **Pages**.
   - Em "Build and deployment" → "Branch", selecione `main` (ou `master`) e a pasta `/ (root)`.
   - Clique em **Save**.

5. **Aguarde 1 a 2 minutos.** O GitHub vai gerar um link público, algo como:

   ```
   https://SEU-USUARIO.github.io/esquetes-indigenas-jogos/
   ```

   Esse link mostra automaticamente o `index.html` (o menu), e os três jogos ficam
   acessíveis em:
   ```
   https://SEU-USUARIO.github.io/esquetes-indigenas-jogos/siaburu.html
   https://SEU-USUARIO.github.io/esquetes-indigenas-jogos/nossa_luta_e_ancestral.html
   https://SEU-USUARIO.github.io/esquetes-indigenas-jogos/tape_mbyapeha.html
   ```

6. **Pronto.** Qualquer pessoa, em qualquer navegador (computador ou celular), pode
   acessar esse link e jogar — não precisa de conta, instalação ou aplicativo.

## Atualizações futuras

Para editar qualquer jogo depois, basta abrir o arquivo `.html` correspondente
diretamente no repositório do GitHub (ícone de lápis "Edit this file"), fazer a
alteração e clicar em "Commit changes". O site publicado atualiza automaticamente
em 1 a 2 minutos.
